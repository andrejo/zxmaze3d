ZXMaze3D

A simple 3D demo for ZX Spectrum computer

Download this project as a .zip file Download this project as a tar.gz file

Overview

This is a simple 3D demo for the ZX Spectrum computer. Use standard W, S, A, D keys to move around.

Download - Binary .TAP file, run this in an emulator (tested under FUSE and EmuZWin).

Live preview

(uses the JSSpeccy2 javascript-based ZX Spectrum emulator)
